# Alex Rivera - Data Visualization Portfolio

A high-tech, interactive portfolio for a Data Visualization Engineer. Built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- **Interactive Background**: Mouse-following glow effect.
- **Preloader**: High-tech loading screen with data node animations.
- **Project Archive**: Modal-based project grid with smooth transitions.
- **Hire Me Modal**: Contact options with interactive icons.
- **Responsive Design**: Mobile-first approach with desktop-first precision.
- **Neon Theme**: Electric Blue and Cyan accents on a dark background.

## Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite 6
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion (motion/react)
- **Icons**: Lucide React

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Environment Variables

- `GEMINI_API_KEY`: Required for GenAI features (if implemented).
